CSS

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It allows you to control the layout, colors, fonts, and overall appearance of your web pages.

Learn more about CSS

Types of CSS

  • Inline CSS: Applied directly to an HTML element using the "style" attribute.
  • Internal CSS: Defined within a style tag in the head section of the HTML document.
  • External CSS: Linked to an external stylesheet file using the tag.
